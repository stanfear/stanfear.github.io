Intelligence Artificielle

Contexte : Projet d'intelligence artificielle de Seconde année de formation d'ingénieur en Cognitique

Technologie : C#, Windows Forms

Description : il s'agit d'un projet effectué en seconde année de formation d'ingénieur pour mes cours d'intelligence artificielle, il contient des algorithmes de classification avec apprentissage supervisé et non supervisé, un algorithme de recherche de chemin (A*), et un algorithme de reconnaissance de formes simples.

Toutes les modifications faites par rapport au sujet original ont été faites dans le but de pouvoir expliquer plus clairement à mon binôme le fonctionnement de certains algorithmes qui, parfois, peuvent être difficiles à comprendre. Ainsi, pour l'algorithme de recherche de chemin avec A*, il est possible de visualiser les nœuds ouverts et fermés, et comparer les étapes de recherche avec un simple algorithme de Dijkstra. Concernant l'algorithme d'apprentissage, la visualisation permet de comprendre pourquoi un point est associé à une classe en affichant les différentes classes de chaque point pouvant exister sur le graphique.

Mon rôle : Ce projet était effectué en binôme, les principales fonctions du projet ont été faites en groupe, mais j'ai effectué divers ajouts par rapport aux consignes originales.

Disponibilité des sources : Les fichiers sources pour Visual studio sont disponibles en téléchargement ici (pensez bien à choisir le bon projet de démarrage) Les fichier exécutables sont également disponibles dans l’archive (Projet IA 2A/<nom-du-projet>/bin/Debug/)

Synthèse d'images 3D

Contexte : Projet de synthèse d'images en 3D, affichage de formes simples en 3D (cylindre, sphère et pavés)

Technologie : C#, Windows Forms

Description : il s'agit d'un projet effectué en seconde année de formation d'ingénieur pour mes cours d'informatique. Dans ce projet de modélisation d'objets dans un espace en 3 dimensions, j'ai fait le choix d'utiliser une technique de projection de rayons pour obtenir le rendu final. Ce choix permet, avec un peu de gymnastique oculaire, de percevoir des images en stéréoscopie.

Mon rôle : Ce projet était effectué en binôme, mon rôle était la réflexion et le développement du moteur graphique

Disponibilité des sources : Les fichiers sources pour Visual studio sont disponibles en téléchargement ici. L'exécutable est également disponible ici.

Question de point de vue

Contexte : Projet de fin d'études de ma formation d'ingénieur en Cognitique

Technologie : C#, Windows Forms, OpenCV

Description : il s'agit d'un projet effectué en dernière année de formation d'ingénieur. Le projet consiste à mettre en place un logiciel permettant de suivre la position de la caméra ayant filmé une vidéo dans un environnement connu, en calculant les déplacements de celle-ci grâce à des images placées dans l'environnement.

Mon rôle : Ce projet était effectué en binôme, ma mission a été de prendre en main la librairie pour ensuite pouvoir l'expliquer à mon binôme. Le reste (développement, réflexion sur l'algorithme) est le résultat d'un effort commun.

Subsurfaces (Mod factorio)

Jeu des 7 familles

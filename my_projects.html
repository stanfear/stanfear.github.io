Artificial intelligence

Context : AI project done during my second year of my Master's degre course

Technology : C#, Windows Forms

Description : This is a project done during my second year of master's degree course for my Artificial intelligence class, it contains classification algorithmes using supervised and non supervised machine learning, A pathfinding algorithme (A*), and a basic pattern recognition algorithme.

All the changes and additions from the instructions had for purpose (other than having fun codding) to explain my co-worker the way some algorithme works, as they can be hard to understand. The result for the pathfinding algorithme was the ability to see the open and closed nodes list drawn on the map, as well as the ability to see the difference between using a heuristic cost or not. For the classification algorithme each points present on the graph was collored depending on the class it was affected to, allowing to clearly see the delimitation of the two classes.

My role : this project was done in pairs, the major part of the work was done by each member equally, but the changes and additions were made by me.

Source code availability : The solution files for Visual Studio can be downloaded here (don't forget to change the starting project). The binairy files can be found in the archive as well (under Projet IA 2A/<project-name>/bin/Debug/)
